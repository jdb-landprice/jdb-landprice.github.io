⚙️ ຕັ້ງຄ່າ Google Sheets
ລະບົບນີ້ເກັບ ແລະ ດຶງຂໍ້ມູນໂດຍກົງຈາກ Google Sheets ຂອງທ່ານ. ຂໍ້ມູນທັງໝົດຈະຖືກບັນທຶກໃສ່ sheet tab ຊື່ "data".
1
ສ້າງ Google Cloud Project + API Key
ໄປທີ່ console.cloud.google.com → ສ້າງ Project → ຄົ້ນ "Google Sheets API" → Enable → APIs & Services → Credentials → + Create Credentials → API Key
2
ສ້າງ / ເປີດ Google Sheet ແລະ ໃສ່ Header
ສ້າງ Google Sheet ໃໝ່ → ສ້າງ tab ຊື່ "data" → ໃສ່ header ແຖວທີ 1:
ID | Province | Price_per_m2 | Area_m2 | Env_Score | Loc_Score | Access_Score | Phys_Score | Level_Score | Infra_Score | HBU_Score | Liq_Score
ຫຼັງຈາກນັ້ນ Share → Anyone with the link (Viewer) ເພື່ອໃຫ້ API Key ອ່ານໄດ້.
ຈາກ URL: docs.google.com/spreadsheets/d/[ID ຢູ່ນີ້]/edit
3
